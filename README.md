# raspberry-kmini
reverse engineering of keurig k-mini using Raspberry Pi, 
the goal is to replace the motherboard of the k-mini (dead) with a Raspberry Pi.

#Components:
DC 12v Motor: "CJWV23-2D12A1"

inputs: lid closed "button"
